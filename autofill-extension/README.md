# chrome-autofill-for-jobs

[Getting Started on Chrome Extensions]("https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world")

# Quickstart

```
git clone https://github.com/nat-leo/chrome-autofill-for-jobs.git
```

In the repository, run the following to install all dependencies
```
npm install
```

Then, build the project.
```
npm run build
```

More documentation, architecture decisions, and roadmaps are available in `/docs`
